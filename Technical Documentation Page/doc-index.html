Introduction

Python is a high-level, interpreted programming language known for its readability and versatility.

It supports multiple programming paradigms including procedural, object-oriented, and functional programming.

Python is widely used in web development, data science, artificial intelligence, and automation.

The simplicity of its syntax makes it an excellent choice for beginners and professionals alike.

Python is cross-platform and open-source, with a massive community that contributes to its libraries and frameworks.

Getting Started

To install Python, visit the official Python website and download the installer for your operating system.

Once installed, you can run Python in the terminal by typing python or python3.

Python files are saved with the .py extension.

To run a Python script, use: python script.py

  • Install Python
  • Check installation with python --version
  • Write your first script

Python comes with a built-in REPL (Read-Eval-Print Loop) for interactive testing.

Data Types

Python supports several built-in data types including numbers, strings, lists, tuples, sets, and dictionaries.

Numbers include integers (int), floating-point (float), and complex numbers.

Strings (str) are sequences of characters enclosed in quotes.

Lists are ordered, mutable collections of items.

Dictionaries store data in key-value pairs.

  • int: 42
  • float: 3.14
  • str: "hello"
  • list: [1, 2, 3]
  • dict: {"key": "value"}
Control Flow

Python provides conditional statements and loops for controlling the flow of a program.

Conditional statements include if, elif, and else.

Loops include for and while.

Python also supports loop control statements such as break and continue.

Example of an if statement:

if x > 0:
  print("Positive")
else:
  print("Non-positive")

Functions

Functions in Python are defined using the def keyword.

They help in structuring code and avoiding repetition.

Functions can accept parameters and return values.

Python also supports lambda (anonymous) functions.

Example of a function:

def greet(name):
  return f"Hello, {name}"

print(greet("Alice"))
